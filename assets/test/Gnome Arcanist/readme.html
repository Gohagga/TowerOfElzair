My entry for the current Modelling Contest.

For clarity, the "Morph" animation is an all-purpose transformation, while Spell Gold/Lumber Alternate were meant to be used with triggers as individual transformations for their respective forms. Considering the way animation tags work, both forms have a Morph animation, allowing the Arcanist to transform back into his base state without the same triggering hassle as it would take to get INTO those forms.
Feel free to use and/or edit this model, so long as you credit me.
